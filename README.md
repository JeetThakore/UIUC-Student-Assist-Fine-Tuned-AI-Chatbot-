# UIUC-Student-Assist-Fine-Tuned-AI-Chatbot-
UIUC Student Assist is a fine-tuned FLAN-T5 chatbot trained on 1,500+ UIUC PDFs and integrated with LangChain, FAISS, and Streamlit. It delivers context-aware answers to student queries, combining fine-tuning with retrieval-augmented generation to cut query resolution time by 70%.
